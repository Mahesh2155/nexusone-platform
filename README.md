# NexusOne Platform

Enterprise Multi-Tenant SaaS Platform for SMEs.

## Overview

NexusOne is a scalable SaaS platform designed to help businesses manage customers, leads, teams, and workflows from a single platform.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
